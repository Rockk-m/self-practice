2.Constructor Function
 function Person(name, age, birthdate, gender) {
    this.name = name
    this.age = age
    this.birthdate = birthdate
    this.gender = gender
}
สร้าง template สำหรับสร้าง object หลายๆ ตัว
ใช้ this เพื่ออ้างอิงถึง object ที่จะถูกสร้าง
ใช้กับ new Person() เพื่อสร้าง instance ใหม่
