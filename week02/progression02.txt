ความรู้ที่ได้

1.เรื่อง Object comparison
เวลาเขียน book1 === book2 มันจะได้ false เสมอ เพราะ === เช็กแค่ reference

2.shallowEquality เช็ค property ชั้นเดียว เช่น key กับ value ต้องตรงกัน
แต่ถ้าเจอ object ซ้อนข้างใน (nested) มันจะเช็กไม่ถึง ต้องใช้ deep equality อีกที

3.การใช้ Object.keys()
Object.keys(obj) จะดึง key ทั้งหมดออกมาเป็น array → ใช้เช็กจำนวน key 
หรือวน loop เพื่อเข้าถึงค่าข้างใน

4.การใช้ for...of กับ array ของ keys
เป็นวิธีที่อ่านง่ายกว่า for...in เพราะมันไม่ลาก inherited properties มาด้วย
